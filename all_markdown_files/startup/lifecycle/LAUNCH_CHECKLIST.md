# LAUNCH CHECKLIST
*Pre-launch verification checklist — nothing ships until all items are checked*

---

## PRODUCT READINESS

### Core Functionality
- [ ] MVP features complete
- [ ] All critical paths tested
- [ ] Error handling in place
- [ ] No white screens of death

### User Experience
- [ ] Mobile responsive
- [ ] Loading states present
- [ ] Error messages clear
- [ ] Empty states handled

### Performance
- [ ] Page load < 3 seconds
- [ ] API response < 1 second
- [ ] No memory leaks

---

## PAYMENT READINESS

- [ ] Payment processor connected
- [ ] Test payments working
- [ ] Webhooks configured
- [ ] Receipts/invoices generating
- [ ] Refund flow tested

---

## SECURITY

- [ ] HTTPS enabled
- [ ] Authentication working
- [ ] Authorization enforced
- [ ] Input validation in place
- [ ] Secrets not in code

---

## ANALYTICS & TRACKING

- [ ] Analytics (Mixpanel/PostHog/GA) installed
- [ ] Key events tracked
- [ ] Funnel setup complete
- [ ] Dashboard accessible

---

## LEGAL & COMPLIANCE

- [ ] Terms of Service published
- [ ] Privacy Policy published
- [ ] Cookie consent (if required)
- [ ] Contact info on site

---

## CUSTOMER SUPPORT

- [ ] Support channel live (email/chat)
- [ ] Help docs in place
- [ ] Onboarding flow ready
- [ ] Response SLA defined

---

## MARKETING READINESS

- [ ] Landing page complete
- [ ] Launch content ready
- [ ] Social posts scheduled
- [ ] Email sequence ready
- [ ] Press list (if applicable)

## OPERATIONAL READINESS

- [ ] Health check endpoint working
- [ ] Monitoring alerts set up
- [ ] Error logging active
- [ ] Backups configured
- [ ] Runbook drafted

## LAUNCH DAY CHECKLIST

### Pre-Launch (T-1 day)
- [ ] Final smoke test
- [ ] Deploy to production
- [ ] Verify production works
- [ ] Notify team

### Launch Day (T-0)
- [ ] Marketing goes live
- [ ] Monitor metrics
- [ ] Support on standby

- [ ] Check for errors
- [ ] Review early metrics
- [ ] Collect feedback
- [ ] Plan next steps
