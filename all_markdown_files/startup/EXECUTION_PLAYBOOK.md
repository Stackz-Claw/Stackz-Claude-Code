# EXECUTION PLAYBOOK
*How Founder runs a venture from 0 to launch*

---

## PHASE 1: INTAKE (Week 1)

### Day 1-2: Receipt
1. Receive approved proposal from Stackz
2. Review proposal details
3. Confirm budget and timeline

### Day 3-5: Setup
1. Create `startup/ventures/[slug]/` workspace
2. Write `VENTURE_BRIEF.md` from proposal
3. Set up metrics tracking
4. Create project in task manager

### Day 6-7: Kickoff
1. Coordinate with Warden for agents
2. Kick off discovery sprint
3. Schedule team introductions

---

## PHASE 2: BUILD (Weeks 2-4)

### Sprint Structure
- 1-week sprints
- Daily standups (async)
- Sprint review at end
- Retrospective

### Team Coordination

| Team | Focus | Frequency |
|------|-------|-----------|
| Forge | MVP dev | Daily sync |
| Canvas | Brand/UI | 2x week |
| Cashflow | Revenue setup | Weekly |

### Milestones
- [ ] Discovery complete
- [ ] MVP scope locked
- [ ] Design complete
- [ ] MVP dev complete
- [ ] Internal testing

---

## PHASE 3: LAUNCH (Week 5)

### Pre-Launch
1. Run LAUNCH_CHECKLIST.md
2. Final testing
3. Content ready
4. Support setup

### Launch Day
1. Deploy to production
2. Monitor metrics
3. Customer support active
4. Marketing goes live

### Post-Launch
1. Monitor for issues
2. Gather initial feedback
3. Track early metrics

---

## PHASE 4: GROW (Weeks 6-12)

### Growth Sprint 1 (Week 6-8)
- Initial acquisition channels
- Landing page optimization
- Onboarding flow

### Growth Sprint 2 (Week 9-12)
- Scale what works
- Test new channels
- Retention focus

---

## PHASE 5: EVALUATE (Monthly)

### Monthly Review
1. Collect metrics
2. Compare to projections
3. Identify issues
4. Recommend actions

### Decision Points
- Continue: On track, metrics good
- Pivot: Key assumptions wrong
- Graduate: Criteria met
- Wind down: Not viable

---

## PHASE 6: GRADUATION

### Graduation Checklist
- [ ] MRR threshold met
- [ ] Churn under control
- [ ] Moat identified
- [ ] Stability ready

### Handoff
1. Create stability portfolio folder
2. Transfer docs
3. Brief Stability Team
4. Close Startup Team tasks
