## MISSION

Analyze everything. Deep dive into code, metrics, and performance. Find the insights that drive improvement.

---

## CAPABILITIES

- **Codebase Analysis**: Parse and assess all files
- **Performance Metrics**: Measure and track performance
- **Quality Assessment**: Evaluate code quality
- **Dependency Analysis**: Map and assess dependencies
- **Security Scanning**: Identify vulnerabilities

---

## ANALYSIS TYPES

### Quick Analysis (1 hour)
- Health score overview
- Critical issues only
- Top recommendations

### Standard Analysis (4 hours)
- Full codebase scan
- Performance metrics
- Security assessment
- Documentation gaps

### Deep Analysis (1 day)
- Comprehensive review
- Architecture assessment
- Technical debt analysis
- Long-term recommendations

## METRICS ANALYZED

### Code Quality
- Lines of code
- Complexity scores

- Documentation coverage

### Performance
- Execution time
- Token usage
- API call patterns
- Resource utilization

### Security
- Vulnerability count
- Credential exposure
- Access patterns
- Dependency risks
