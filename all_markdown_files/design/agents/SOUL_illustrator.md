## ASSET TYPES

### Logos
- Primary logo (full color)
- Monochrome variants (black, white)
- Icon-only variant
- File formats: SVG (primary), PNG, AI/EPS

### Illustrations
- Style: [defined per venture brand]
- Use cases: Landing pages, blog posts, emails, presentations
- File formats: SVG, PNG (various resolutions)

### Social Assets
- Platform-specific sizes:
  - Twitter/X: 1200x675 (card), 1080x1080 (post)
  - LinkedIn: 1200x627 (post), 1080x1080 (post)
  - Instagram: 1080x1080 (post), 1080x1920 (story)
  - Facebook: 1200x630 (post)
- File formats: PNG, JPG

### Ad Creative
- Standard IAB sizes:
  - Leaderboard: 728x90
  - Medium Rectangle: 300x250
  - Wide Skyscraper: 160x600
  - Large Rectangle: 336x280
  - Mobile Banner: 320x50
- Static and animated variants

---

## QUALITY STANDARDS

### All Assets Must Have
- [ ] Brand-consistent colors
- [ ] Brand-consistent typography
- [ ] Proper resolution for intended use
- [ ] Accessibility: alt text for all images
- [ ] File size optimized (<500KB for web)

### Logo Requirements
- [ ] Vector format (SVG)
- [ ] Works at small sizes (32x32 minimum)
- [ ] Works on light and dark backgrounds
- [ ] Variations for different contexts

---

## EXPORT SPECIFICATIONS

### Web Assets
- Format: PNG (with transparency where needed) or SVG
- Compression: Optimized for web (TinyPNG, SVGO)
- Naming: `[asset-name]-[width]x[height].[ext]`

### Print Assets
- Format: AI, EPS, or high-res PDF
- Color: CMYK
- Resolution: 300 DPI minimum

### Social Media
- Format: PNG or JPG
- Resolution: 2x for retina displays
- Naming: `[platform]-[asset-name].[ext]`
