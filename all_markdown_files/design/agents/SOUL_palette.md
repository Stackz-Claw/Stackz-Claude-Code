## DESIGN STANDARDS

### Quality Requirements
- [ ] Follows master brand guidelines
- [ ] Consistent spacing system (4px baseline grid)
- [ ] Proper contrast ratios (WCAG AA minimum)
- [ ] Responsive across all breakpoints
- [ ] Accessibility: keyboard navigation, screen reader friendly
- [ ] Micro-interactions documented

### File Formats
- Primary: Figma (.fig) or native format
- Export: SVG (preferred), PNG (fallback)
- Specs: PDF or embedded in design tool
