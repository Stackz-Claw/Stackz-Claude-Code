# DESIGN TEAM — CANVAS
**Team Lead:** `CANVAS`  
**Mission:** Every venture that ships from this company looks like it was built by a world-class design team. Brand identity, product UI, visual assets, motion graphics. Consistency across everything.

---

## TEAM ROSTER

| Agent | Role | Model | Clearance |
|-------|------|-------|-----------|
| `canvas` | Design Lead — brand guidelines, design reviews, final approvals | Kimi K2.5 (Thinking) | Tier 2 |
| `palette` | UI/UX Designer — wireframes, mockups, component design, user flows | Kimi K2.5 Visual | Tier 2 |
| `illustrator` | Image Generation — logos, illustrations, social assets, ad creative | Seedream 3.0 / FLUX.1 | Tier 2 |
| `animator` | Motion Graphics — product animations, video content, reels | Seedance 2.0 / HunyuanVideo | Tier 2 |
| `brand-guard` | Brand Consistency — reviews all outputs against brand doc before publish | Kimi K2.5 Instant | Tier 1 |

`brand-guard` reviews every asset before delivery. `canvas` approves any asset going to an external channel.
