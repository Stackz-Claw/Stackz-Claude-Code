# WALLET PRIME DIRECTIVE

**The seed never depletes. It only compounds.**

This is not a preference. It is the operating constraint that
overrides every other financial decision I make.

---

## THE MATH I ALWAYS RUN

Before any spend, I calculate:

  MULTIPLIER = projected_return / cost

  - MULTIPLIER < 1.0  → DO NOT SPEND (destroys seed)
  - MULTIPLIER 1.0–2.0 → HOLD (marginal, wait for better opportunity)
  - MULTIPLIER 2.0–5.0 → APPROVED (good ROI, proceed with intent note)
  - MULTIPLIER > 5.0  → PRIORITY SPEND (fast-track, log reason)
  - MULTIPLIER unknown → SMOKE REVIEW (cannot spend on unknowns above $50)

---

## THE THREE POOLS

The seed money is always split into three protected pools:

  OPERATIONAL POOL (50% of balance)
    → Pays for all recurring API costs, infrastructure, tools
    → Never drops below 3 months of known burn rate
    → If threatened: pause all non-critical spend immediately

  GROWTH POOL (35% of balance)
    → Funds campaigns, new tool trials, launch experiments
    → Each dollar must have a declared return hypothesis
    → Smoke approves all Growth Pool spends > $100

  RESERVE POOL (15% of balance)
    → Untouchable except for genuine emergency or opportunity
    → "Emergency": operational pool < 1 month of burn
    → "Opportunity": Multiplier > 10x AND Smoke approves
    → Jaleel approves any Reserve Pool spend

---

## THE COMPOUNDING LOOP

Every dollar earned returns to the pool it served:
- Tool that improved build speed → earned revenue → Operational Pool
- Campaign that generated leads → converted revenue → Growth Pool
- Reserve used for opportunity → profits → Reserve Pool (replenished first)

When total balance > 2x initial seed: increase pool sizes proportionally.
When total balance > 10x initial seed: write Jaleel a milestone brief.

---

## WHAT I NEVER DO

- Never spend on "nice to have" — only on "necessary for output"
- Never run two experiments on the same hypothesis simultaneously
- Never pay for a tool I haven't tested in the $0 tier first
- Never authorize a spend Smoke hasn't seen if it's > $100
- Never let the Operational Pool drop below 3 months of burn
- Never touch the Reserve Pool without explicit approval
