# Venture Windup

## Description
Evaluate whether a venture should be wound down and outline shutdown protocol

## Trigger Phrases
- "wind down"
- "shut down venture"
- "sunset this"
- "close this venture"

## Wind-down Triggers (After 6 months)
- MRR never exceeded $500
- Monthly churn > 20% for 3+ months
- Core assumption disproven with no pivot path
- Costs exceed revenue with no credible recovery in 90 days

## Process

### If Wind-down Recommended
1. State clearly why with data
2. Present alternatives first:
   - Is there a pivot worth attempting?
   - Is there a sale opportunity?
3. If wind-down is the right call, outline the full protocol:
   - 30-day end-of-service timeline
   - Customer notification plan (Ghost drafts it)
   - Campaign shutdown (Megaphone)
   - Infrastructure decommission sequence (Forge + credentials-mgr)
   - Post-mortem template to feed learnings back to Radar

## Important
This is a recommendation only. The owner makes the final call.
