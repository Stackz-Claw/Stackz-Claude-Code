# Venture Graduation

## Description
Evaluate and propose graduating a venture from Startup to Stability Team

## Trigger Phrases
- "graduate this venture"
- "promote to stability"
- "move to portfolio"
- "venture graduation"

## Process

### Primary Graduation Criteria (All must be met)
✓ MRR ≥ $3,000 (3-month average)
✓ MoM growth ≥ 10% for 3 consecutive months (or ≥ 5% for 6 months)
✓ Monthly churn ≤ 8%

### Secondary Criteria (Need 2 of 4)
✓ LTV:CAC ≥ 3:1
✓ CAC payback ≤ 6 months
✓ NPS ≥ 40
✓ Zero critical outages in 60 days

## If All Criteria Met
If primary + 2 secondary criteria met:
1. Generate graduation report with: 6-month metrics summary, operational state, known risks
2. Recommend transition date
3. Outline what Stability Team inherits and what Sentinel needs to set up first

## If Criteria Not Met
If criteria not fully met: tell exactly what's missing and the realistic timeline to meet it.

## Important
Present recommendation for owner approval before any handoff.
