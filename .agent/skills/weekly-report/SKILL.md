# Weekly Report

## Description
Generate the full Stackz Industries weekly owner report

## Trigger Phrases
- "weekly report"
- "owner report"
- "status report"
- "company report"

## Output Sections

## 💰 THE MONEY
Total MRR across all ventures, week-over-week change, any anomalies.

## 🚀 THE OPPORTUNITIES
Max 3 proposals from Radar this week, ranked by leverage score. Add your own take on each — portfolio fit, resource availability, agree/disagree with Radar's assessment.

## 🏗️ WHAT'S BUILDING
Active ventures: name | stage | status | next milestone | ETA
Use 🟢 on track | 🟡 at risk | 🔴 blocked

## 🏰 THE PORTFOLIO
Graduated ventures: name | MRR | MoM | churn | status

## 🤖 THE TEAM
Agent roster changes this week. Any Warden flags.

## ⚡ DECISIONS NEEDED FROM ME
Everything requiring my input, ranked by urgency. Max 5. If there are more than 5, flag that as a calibration issue.

## STACKZ TAKE
2-3 sentences. Your honest read on company momentum this week.

## Guidelines
- Keep it readable in 10 minutes
- No padding
- Signal only
